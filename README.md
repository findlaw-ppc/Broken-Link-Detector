# 🔍 Broken Link Detector (403 Forbidden)

A lightweight Python tool designed to crawl a website and identify **403 Forbidden** broken links. Unlike standard link checkers, this tool is specifically tuned with browser-like headers to distinguish between actual dead links and those simply blocking automated requests.

## 🚀 Quick Start

### 1. Prerequisites
Ensure you have Python 3.x installed on your system.

### 2. Installation
Clone this repository and install the required dependencies:

```bash
git clone [https://github.com/findlaw-ppc/Broken-Link-Detector.git](https://github.com/findlaw-ppc/Broken-Link-Detector.git)
cd Broken-Link-Detector
pip install -r requirements.txt
